# 理论验证：模型预测 vs 实证文献

## 1. 乐观先验效应的数学推导

### 1.1 信号检测理论解释

设智能体的先验信念为 Beta(α, β)，期望合作率为：
$$E[p] = \frac{\alpha}{\alpha + \beta}$$

当施加偏差 bias = δ 时，True标签组的**实际**合作率变为：
$$p_{true} = p_{base} + \delta$$

智能体需要检测的**信号**是 True 组与 False 组的差异：
$$\Delta p = p_{true} - p_{false} = \delta$$

### 1.2 贝叶斯更新中的相对惊讶

关键洞察：贝叶斯更新的强度取决于**相对惊讶**，而非绝对差异。

对于观察到合作事件，似然比为：
$$LR = \frac{P(coop | institutional)}{P(coop | neutral)} = \frac{p_{expected} + \delta}{p_{expected}}$$

**悲观者** (E[p] = 0.25)：
$$LR_{pessimist} = \frac{0.25 + 0.10}{0.25} = 1.40 \quad (+40\% 惊讶)$$

**乐观者** (E[p] = 0.75)：
$$LR_{optimist} = \frac{0.75 + 0.10}{0.75} = 1.13 \quad (+13\% 惊讶)$$

### 1.3 结论

$$\text{信号强度} \propto \frac{\delta}{E[p]}$$

乐观先验使分母更大，信号相对更弱，因此更难触发模型切换。

**验证结果**：
| 先验 E[p] | 理论预测 | 实验结果 |
|-----------|----------|----------|
| 0.17 (极悲观) | 信号强 (δ/0.17 = 0.59) | 20% SF ✓ |
| 0.50 (中性) | 信号中 (δ/0.50 = 0.20) | 18% SF ✓ |
| 0.75 (乐观) | 信号弱 (δ/0.75 = 0.13) | 2% SF ✓ |
| 0.83 (极乐观) | 信号极弱 (δ/0.83 = 0.12) | 0% SF ✓ |

## 2. 与实证文献的对比

### 2.1 Yamagishi (1999) - 高信任者的脆弱性

**文献发现**：
> "Trust, Gullibility, and Social Intelligence" (Yamagishi, Kikuchi, & Kosugi, 1999)
> - 高信任者在PD中合作率82%，低信任者49%
> - 高信任者更容易被不可信的人利用
> - 但匿名条件下，高信任者欺骗检测更准确

**模型预测 vs 文献**：

| 现象 | Yamagishi发现 | 我们模型预测 | 匹配？ |
|------|--------------|--------------|--------|
| 高信任者更合作 | ✓ (82% vs 49%) | ✓ (乐观者初始合作高) | ✅ |
| 高信任者更易被利用 | ✓ | ✓ (乐观者制度采纳低) | ✅ |
| 高信任者检测准确(匿名) | ✓ | ❓ 需要测试 | 待验证 |

**关键洞察**：
Yamagishi的"gullibility"概念 ≈ 我们的"信号检测失败"
- 高信任者预期好结果 → 好结果不惊讶 → 不更新信念
- 低信任者预期坏结果 → 好结果惊讶 → 更新信念

### 2.2 Balliet et al. (2014) - 内群偏好元分析

**文献发现**：
> "Ingroup Favoritism in Cooperation: A Meta-Analysis"
> - 效应量 d = 0.32 (小到中等)
> - 内群偏好而非外群贬低
> - 互惠预期驱动

**模型预测 vs 文献**：

| 现象 | 元分析发现 | 我们模型 | 匹配？ |
|------|-----------|---------|--------|
| 效应量 | d = 0.32 | 合作差 ~5-15% | ✅ 数量级一致 |
| 内群偏好主导 | ✓ | ✓ (True组合作↑) | ✅ |
| 互惠预期机制 | ✓ | ✓ (EFE互惠项) | ✅ |
| 任意群体可触发 | ✓ (MGP) | ✓ (随机标签) | ✅ |

**效应量校准**：
- 文献 d = 0.32 ≈ 合作率差异 ~10-15%
- 我们模型：bias=0.10 → 合作差 5-15%
- **数量级匹配！**

### 2.3 Lane (2016) - 第三方博弈

**文献发现**：第三方独裁者博弈最容易产生内群偏好

**模型启示**：我们的模型是直接互动，可能低估了观察学习的效应

## 3. 模型预测的新假说（可实验验证）

基于模型，我们预测以下可测试假说：

### H1: 先验-敏感性权衡
> 悲观者对正向信号更敏感，乐观者对负向信号更敏感

**实验设计**：
- 测量被试先验信任水平
- 让其玩含正向/负向偏差的MGP博弈
- 预测：低信任者更快检测正向偏差，高信任者更快检测负向偏差

### H2: 种群规模-临界质量
> 小群体（8-16人）比大群体（64+人）更容易形成群体规范

**实验设计**：
- 不同规模群体的MGP实验
- 测量规范形成时间和强度
- 预测：小群体更快达成共识，但方差更大

### H3: 制度的"空壳化"
> 长期互动后，群体标签持续存在但行为差异消失

**实验设计**：
- 长期MGP实验（多轮）
- 跟踪标签认同 vs 行为差异
- 预测：认同持续，行为趋同
